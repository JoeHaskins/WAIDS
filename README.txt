## Project title
Web Application Information Disclosure Scanner (WAIDS)

## How to install and run the Application
Download the Project folder from github.
Located here: https://github.com/JoeHaskins/WAIDS
Open the folder with command prompt and compile the Java files using the command:

javac Main.java Logic.java Frame.java VulnData.java

Run the program using:

java Main
